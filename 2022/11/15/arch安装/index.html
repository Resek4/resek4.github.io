fdisk

cfdisk

lsblk
可以看当前的硬盘分区情况

创建引导分区，并且格式化为
mkfs.fat -F32 /dev/引导分区

mkfs.ext4 /dev/根分区

创建/mnt/boot
挂载引导分区到/mnt/boot
挂载根分区到/mnt

改变镜像源
pacstrap /mnt base base-devel linux linux-firmware dhcpcd

配置自动挂载fstab
原本是空的，现在使用gen可以将当前挂载情况写到fstab上

使用arch-chroot /mnt
进入已经在根分区搞好的系统

设置时区
ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
hwclock –systohc

安装必要安装包
pacman -S vim dialog wpa_supplicant ntfs-3g networkmanager netctl

设置语言选项
vim /etc/locale.gen
去掉zh_CN的各个utf8
en_US的utf8

locale-gen

创建vim /etc/locale.conf
写入（设置）默认语言
LANG=en_US.UTF-8

设置/etc/hostname
/etc/hosts

127.0.0.1 localhost
::1 localhost
127.0.1.1 myhostname.localdomain myhostname

passwd

双系统需要安装启动项
pacman -S os-prober grub efibootmgr

vim /etc/default/grub
取消最后一行
GRUB_DISABLE_OS_PROBER=false
的注释

grub-mkconfig -o /boot/grub/grub.cfg

进入系统

建立用户

设置密码

找到 # %wheel 回车这行，删除注释

技术|初级：如何在 Arch Linux 中安装 Yay AUR 助手

Display Manager
